# Cursor Notes

---

## Project Structure

- `LLD/` contains low-level design preparation notes and coding exercise material.
- `GIT/` is the Git mastery journal section with deep conceptual docs and practice labs.
- `frontend-mastery/` is the senior frontend mastery track focused on SPA internals, JavaScript runtime behavior, reactivity, React/Next/Vue trade-offs, frontend architecture, UI/UX craft, debugging, testing, and review judgment.

---

## Git Journal Notes

- The Git docs are organized around the senior mental model: working tree, index, object store, refs, reflog, and remotes.
- Practice material is split into spoiler-free labs and a separate answers file.

---

## Frontend Mastery Notes

- The frontend docs intentionally avoid framework encyclopedias and focus on foundational internals: browser pipeline, event loop, rendering, reactivity, lifecycle, server/client boundaries, state ownership, data placement, performance, UI/UX states, and debugging instincts.
- Practice material is split into thought exercises and a separate answer file to build senior review judgment.
